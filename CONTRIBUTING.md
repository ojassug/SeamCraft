# Contributing

Thank you for taking the time to contribute to SeamCraft.

## Development Workflow

- Keep commits small.
- Make one feature per commit.
- Test before committing.

## Commit Message Examples

- Initial project setup
- Add image loading
- Implement energy map
- Build graph representation
- Implement Dijkstra algorithm
- Animate seam removal
- Improve documentation

## Coding Style

- C++17
- Separate headers and source files
- Clear variable names
- Small focused functions
- Comment algorithms, not every line

## Branch Strategy

`main`

Future contributors may use feature branches.
